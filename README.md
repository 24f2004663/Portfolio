# Manoj K. Portfolio

## Setup

This project uses Next.js 15.

### Prerequisites

Ensure Node.js 18+ is installed.
If `npm` is not found, you can use the local version installed in `~/.gemini/node`:

```bash
export PATH=$HOME/.gemini/node/bin:$PATH
```

### Running Locally

```bash
npm install
npm run dev
```

### Building

```bash
npm run build
npm start
```
